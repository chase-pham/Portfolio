Portfolio

Welcome to my portfolio! Below are some of the projects I've had the pleasure to work on.

Online Bookstore Website

This project involved designing and developing a responsive website for an online bookstore. It utilized HTML5, CSS3, and JavaScript for the front end, and Node.js with MongoDB for the back end. The outcome was a user-friendly platform that allowed customers to browse, search, and purchase books online.

E-commerce Mobile App

As part of a team, I contributed to building an e-commerce mobile app for iOS and Android platforms. The app was developed using React Native, providing a native-like experience for users. Features included product listings, user authentication, and secure payment processing. The project resulted in a successful launch and positive user feedback.
